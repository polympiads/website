This website presents **Polympiads**, a student association founded by EPFL students passionate about mathematics and programming.

The site provides general information about the association, its mission, and its activities. Polympiads organizes weekly training sessions at the **Bernoulli Center**, bringing together motivated and talented students to prepare for national and international competitions.

Use this website to:

* Learn what Polympiads is about
* Discover training sessions and events
* Find useful information for current and prospective members

The website serves as the main reference point for anyone interested in joining or learning more about Polympiads.
